 Centrallix....

Welcome to the new website for the Centrallix application server! Centrallix is a web-based app server and data management engine featuring data abstraction, structural embedding, "pro"-DHTML generation, a SQL engine for multi-source queries, and object-based development. That's the official definition... unofficially, we're attempting to build the ultimate web-based application server in terms of flexibility, structure, extensibility, usability for real-world business applications (as opposed to just websites), and perhaps more....


Project Status....

Centrallix currently consists of some 70,000 lines of highly modular C code, and depends heavily on JavaScript for the operation of the generated web pages. The system has many working components (good enough for a small demo), but is still lacking in some very key areas. The project was conceptualized in 1997 and has been underway off and on since late 1998. The main development platform is Linux, although it has been successfully run on non-UN*X platforms via support libraries and/or emulation.
