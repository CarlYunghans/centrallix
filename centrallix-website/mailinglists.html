Centrallix-Announce....

The Centrallix-Announce mailing list is a good list to subscribe to if you're interested in receiving occasional updates about new versions, security fixes, and similar matters. It is a moderated list.

To subscribe, visit http://lists.sourceforge.net/lists/listinfo/centrallix-announce.
To review the list archives, visit http://www.geocrawler.com/redir-sf.php3?list=centrallix-announce.

Centrallix-Users....

The Centrallix-Users mailing list is intended for the discussion of developing applications on top of the Centrallix platform. It is also a good place to ask questions about Centrallix if you're just getting started with it. It is a non-moderated list.

To subscribe, visit http://lists.sourceforge.net/lists/listinfo/centrallix-users.
To review the list archives, visit http://www.geocrawler.com/redir-sf.php3?list=centrallix-users.

Centrallix-Devel....

The Centrallix-Devel mailing list is for discussion of the development of the Centrallix Application Server System itself. If you are interested in writing new DHTML widget modules, ObjectSystem drivers, or core components of Centrallix, this is the list to subscribe to. It is a non-moderated list.

To subscribe, visit http://lists.sourceforge.net/lists/listinfo/centrallix-devel.
To review the list archives, visit http://www.geocrawler.com/redir-sf.php3?list=centrallix-devel.
