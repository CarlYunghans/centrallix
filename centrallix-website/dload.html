Download Centrallix...

To obtain the most recent release copy of Centrallix, visit our

SourceForge downloads page for Centrallix

You MUST download at least the "centrallix", "centrallix-lib", and "centrallix-os" packages, otherwise things will not work! Build and install centrallix-lib and centrallix-os before centrallix.

Be sure to read the release notes that comes with the download! Please note that "Development" (0.7.x, 0.9.x) releases should be considered buggy and unstable with incomplete security frameworks.

Prebuilt RPM downloads have been tested on several RedHat/Fedora type Linux platforms. TGZ files do contain a "spec" file suitable for building RPMs.

Ideally, only "Stable" releases (1.0.x) should be used in production operations.
