Brief Release Notes for 0.9.0


1. This is a development release! Lots of things are still broken (if you hadn't noticed)! :)

2. Centrallix 0.9.0 has full support for Mozilla/Firefox, but only partial support for Internet Explorer. The legacy support for Netscape 4.x is now officially deprecated.


For More Help...


If you need additional help in configuring or using your new Centrallix setup, there are several places that you can go:
  • Online Documentation: For further information about Centrallix, please see: our online documentation site.
  • Centrallix-users Mailing List: We have a mailing list specifically for Centrallix users. To subscribe, see the www.centrallix.net website.
  • Documentation Packages: If you didn't install them, install the centrallix-doc package for more information about application development, and the centrallix-sysdoc package if you are a C/JavaScript coder and want to help on Centrallix itself.
