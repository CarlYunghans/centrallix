chart

type: widget/chart

visual: yes

Overview:

A chart widget is used to display charts and graphs of data. Uses the CHart.js library.

Properties:

Property Type Description
bgcolor string A color, RGB or named, to be used between chart cells and rows.If neither bgcolor nor background are specified, the chart is transparent.
chart_type string Currently supported types are "bar", "line", "radar", and "pie". Defaults to "bar". Chart.js supports much more than this, so new types could be added with relative ease.
objectsource string The name of the ObjectSource widget which will supply data for the chart. We recommend that you do not specify this directly but instead give the chart widget a parent ObjectSource. If not specified, the chart will look for an ObjectSource in its parents.
title string The title of the chart. Default is none.
titlecolor string A color, RGB or named, of the chart title.
width integer width, in pixels, of the chart. The height is determined dynamically.
x integer x-coordinate of the upper left corner of the chart. Default is 0.
y integer y-coordinate of the upper left corner of the chart. Default is 0.

Child Properties:

chart-series

Property Type Description
color string A color, RGB or named, for the chart element (ex: bar) which represents this series.

chart-axis

Property Type Description
label string A label for the axis. Default is none.

Sample Code:

		
$Version=2$
// Here is a simple chart.
chart "widget/chart"
    {
    x=0; y=0;
    width=483;
    chart_type="line";

    series1 "widget/chart-series" {color="green";}
    series2 "widget/chart-series" {color="blue";}

    x_axis "widget/chart-axis" {label="X";}
    y_axis "widget/chart-axis" {label="Y";}

    }
