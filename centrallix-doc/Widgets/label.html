label

type: widget/label

Overview:

The label will simply display some text at certain coordinates with certain properties.

Usage:

The label can be placed inside any visible container, but only nonvisual widgets can be placed within it.

Properties:

x integer x-coordinate of the upper left corner of the label relative to its container
y integer y-coordinate of the upper left corner of the label relative to its container
width integer width, in pixels, of the label
height integer height, in pixels, of the label
text string message that is displayed in the label
align string the alignment of the text. This can be "left", "right", or "center"
background string A background image for the label.
bgcolor string A color, RGB or named, to be used as the label's background. If neither bgcolor nor background are set, the label will be transparent.

Child Properties:

none currently available

Actions:

none currently available

Events:

none currently available

Sample Code:

		
$Version=2$
MyButton "widget/label"
	{
	x = 10; y = 10;
	width = 100;
	height = 15;
	align = "center";
	text = "Test Label";
	}
